# Goal App Backend

Backend API for a goal-tracking application built with Quarkus.

The application supports individual goals and personal challenges.

## Tech Stack

- Java 21
- Quarkus 3.21.4
- Maven
- Hibernate ORM Panache
- H2 Database
- Jakarta REST
- Hibernate Validator
- JUnit
- REST Assured

## Requirements

- Java 21+
- Maven
- WSL/Linux shell or compatible terminal

## Running Locally

```bash
mvn quarkus:dev
